# Naming Conventions — 12 Mythical Signs Posters

## File Naming Standard

All poster files follow this convention:

```
[number]-[sign-name].png
```

### Format Breakdown

- **Number**: Two-digit, zero-padded (01-12)
- **Sign Name**: Lowercase, single word (matches zodiac sign name)
- **Extension**: `.png` (Portable Network Graphics)

### Examples

```
01-aries.png
02-taurus.png
03-gemini.png
04-cancer.png
05-leo.png
06-virgo.png
07-libra.png
08-scorpio.png
09-sagittarius.png
10-capricorn.png
11-aquarius.png
12-pisces.png
```

## Folder Structure

```
12-MYTHICAL-SIGNS-POSTERS/
└── posters/
    └── [numbered files in zodiac order]
```

## Naming Rules

✅ **DO:**
- Use lowercase for sign names
- Use zero-padded two-digit numbers
- Use `.png` extension
- Follow zodiac chronological order
- Keep filenames consistent across the collection

❌ **DON'T:**
- Use spaces in filenames (use hyphens)
- Mix capitalization (keep lowercase)
- Use abbreviations (write full sign names)
- Skip numbers or use non-sequential numbering
- Include special characters (only alphanumeric and hyphens)

## Special Cases

### Multiple Versions
If you have multiple versions of the same sign:
- Primary version: `09-sagittarius.png`
- Alternative version: `09-sagittarius-alt.png` or `09-sagittarius-v2.png`

### Future Formats
If exporting additional formats:
- High-res: `01-aries.png` (main)
- Print-ready: `01-aries-print.png`
- Web: `01-aries-web.jpg`
